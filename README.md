# 랜덤 평가자 배정기

입력한 사람 목록을 기준으로 각 사람마다 본인을 제외한 평가자를 원하는 인원만큼 랜덤 배정하는 React 웹앱입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 Vite가 안내하는 주소로 접속하면 됩니다.

## 빌드

```bash
npm run build
```

## 주요 기능

- 사람 이름 줄바꿈/쉼표 입력 지원
- 중복 이름 자동 제거
- 각 사람마다 본인을 제외한 평가자 랜덤 배정
- 사람별 배정 인원 설정
- 최소 인원 검증
- 재배정
- 결과 복사
- 모바일 반응형 UI

## 주의

현재 방식은 각 사람별로 독립적으로 설정한 수만큼 평가자를 무작위 선택합니다.
따라서 특정 사람이 평가자로 여러 번 배정될 수 있습니다.
